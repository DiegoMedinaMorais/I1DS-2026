# 🚀 SENAI Jaú - Landing Page (Cursos de Tecnologia)

Esta é uma landing page moderna, responsiva e de página única desenvolvida para promover os cursos da área de tecnologia do **SENAI Jaú**. O projeto foi construído focando em alta conversão, utilizando **HTML5** para a estrutura e **Tailwind CSS** para uma estilização rápida, limpa e atraente.

---

## 🎯 Objetivos do Projeto

* **Captação de Alunos:** Apresentar de forma clara e objetiva os cursos de tecnologia disponíveis.
* **Navegação Fluida:** Garantir que o usuário encontre todas as informações essenciais em uma única página (*Single Page Application*).
* **Design Responsivo:** Proporcionar uma experiência de leitura e navegação perfeita tanto em computadores quanto em dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **Tailwind CSS:** Framework CSS baseado em classes utilitárias, garantindo estilização moderna sem a necessidade de arquivos CSS externos pesados.

---

## 💻 Seções da Landing Page

* **Header / Navegação:** Links rápidos para as seções da página e logotipo do SENAI Jaú.
* **Hero Section:** Banner principal com uma frase de impacto e um botão de Chamada para Ação (CTA) em destaque para inscrição.
* **Vitrine de Cursos:** Grade de cards apresentando os cursos de tecnologia (ex: Desenvolvimento Web, Análise de Dados, Redes de Computadores, Inteligência Artificial).
* **Por que o SENAI?:** Seção institucional destacando a infraestrutura, qualidade dos laboratórios e peso do certificado no mercado de trabalho.
* **Depoimentos / Prova Social:** Relatos de ex-alunos que se recolocaram no mercado de trabalho após se formarem no SENAI Jaú.
* **Formulário de Contato:** Área simples para o interessado tirar dúvidas ou demonstrar interesse nas vagas.
* **Rodapé (Footer):** Informações de contato, endereço da unidade de Jaú e links para redes sociais.
